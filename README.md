A modern, responsive, and front-end web application for **Rainbow Welding Works**, a top-rated fabrication and welding service business located in Vijayawada, Andhra Pradesh.

## 🌟 Project Overview

This project serves as the official online presence for Rainbow Welding Works, enabling customers to discover services, view previous work, and book appointments directly. The website is designed to be highly user-friendly, visually appealing, and optimized for local business presentation.

<br />

## 🚀 Features

- **📱 Fully Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.
- **🛠️ Comprehensive Service Catalog**: Showcases core offerings including:
  - Gate & Grill Works
  - SS Fabrication
  - Arc Welding
  - Staircase Works
  - Repair & Maintenance
- **📅 Online Booking System**: A dedicated interface (`booking.html`) for hassle-free appointment scheduling.
- **🖼️ Project Gallery**: Visual portfolio (`gallery.html`) demonstrating craftsmanship.
- **⭐ Customer Trust**: Prominent display of user testimonials and ratings.
- **🗺️ Integrated Maps & Contact**: Direct integration with Google Maps and click-to-call links using the business number (**99080 73025**).

## 💻 Setup and Installation

To run this project locally on your machine, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/rainbow-welding.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd rainbow-welding
   ```

3. **Open the project:**
   Simply open the `index.html` file in your favorite web browser, or use a tool like VS Code Live Server for hot reloading.
